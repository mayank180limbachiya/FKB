WN69 I/O Link i DO ALARM (CHn Gxx Syy zz : PMCm Ybbb = **H )


About Alarm:

This message indicates that the DO alarm occurs at "zz" byte of slot "yy" of group "xx" in channel "n", Y "bbbb" address of P MC path "m". The "**H" shows some bits at which the alarm occurs by hexadecimal. See the descriptions of each unit in this section for the correspondence between "zz"byte of slot "yy" in that message and where alarm occurs in each unit.

In addition, information of status alarm including DO alarms is also output to the system relay area. For details of the system relay area, refer to the FANUC Series 0i-MODEL F CONNECTION MANUAL (HARDWARE) (B-64603EN).


Input:


Alarm Enter:
n xx yy zz m bbb **
WN69 I/O Link I DO ALARM CH( G S :PMC Y = H)

Select Module:

Output:

# Channel:(n) Group:(xx) Path:(m) Check Bits:(bbb) fault location at:
1 _ _ _ _ _

Details of module:


1. I/0 Module for Connector Panel 2. I/0 Module for Operator's Panel (Support Matrix input) 3. I/0 Module for Operator's Panel and Power Magnetics Cabinet 4. I/0 Module Type-2 for Connector Panel
5. Terminal Type I/0 Module 6. Machine Operator's Panel (back side) 7. Operator's Panel Connection Unit (A , B Included) 8. Safety I/0 unit
9. I/0 module for operator's panel supporting safety function 10. I/0 Unit for Power Magnetics Cabinet
