SV401 — VRDY OFF Alarm

About this Alarm

This alarm fires when servo ready signal VRDY does not turn ON or turns OFF during operation. PRDY (position control) was ON while VRDY (velocity control) was OFF. Use DGN 358 to find which signal went OFF.

DGN 358 Input

Invalid — enter a number between 0 and 65535.

How to Read DGN 358

Convert DGN 358 to binary and check bits 5–14 from the lowest upward. Bits are set to 1 sequentially as the amplifier excites. The first bit that is 0 is the processing that failed — that signal caused the VRDY-OFF alarm.
